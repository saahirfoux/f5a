# Unstuck

Short write-ups of real problems we hit on this project and what fixed them. Skim the list below or search the folder for a keyword (e.g. `Expo`, `Gradle`, `Metro`).

## How to add an entry

1. Create a new file: **`kebab-case-topic.md`** (one theme per file; split if it gets long).
2. Use this shape at the top:
   - **Symptoms** — what you saw (errors, logs).
   - **Cause** — why it happens (one paragraph).
   - **Fix** — exact commands or settings.
   - **See also** — links to issues, docs, or other files in this folder.
3. Add one line to the index table below.
